# AMASA Monitor Specification

## Purpose

AMASA(Akeno Mini Air Shower Array)のデータをグラフにし、表示するためのシステム

## Input data

amasa data 

## Monitoring targets

- any1,2,3,4
each channel's countrate

## Environment

- Lab server: Linux + CERN ROOT
- Local: Windows 11 + WSL2 Ubuntu
- Languages: Python, C++ ROOT / PyROOT / uproot as needed
